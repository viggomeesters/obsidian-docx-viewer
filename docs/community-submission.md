# Community submission notes

- Plugin id: `docx-viewer`
- Name: `DOCX Viewer`
- Repository: `https://github.com/viggomeesters/obsidian-docx-viewer`
- Release version: `0.1.0`
- Supported extension: `.docx`
- Scope: read-only inspection of document text, headings, tables, comments, notes, media metadata, and package diagnostics.

## Manual review points

- Release tag must be exactly `0.1.0`.
- Release assets must include `main.js`, `manifest.json`, and `styles.css`.
- The plugin must not claim `.doc`, `.docm`, or `.dotm` support.
- The plugin must not launch external apps, write source files, execute embedded content, or use network APIs.
